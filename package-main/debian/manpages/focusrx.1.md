# NAME
focusrx - System library and installation checker

# SYNOPSIS
**focusrx**

# DESCRIPTION
**focusrx** is a command-line utility that fixes Kubuntu Focus laptops for Nvidia drivers and other items to make sure library conflicts are avoided.

# OPTIONS
none
