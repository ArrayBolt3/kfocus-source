# NAME
shrinkpdf - Reduces PDF filesize by lossy recompressing with Ghostscript.

# SYNOPSIS
**shrinkpdf**

# DESCRIPTION
**shrinkpdf** reduces PDF filesize by lossy recompressing with Ghostscript.
Not guaranteed to succeed, but usually works.

# USAGE
**shrinkpdf** infile [outfile]
